The fantastic fonts of Google and how to use them (Luckiest Guy)

Import Google's font externally (Marko One)

Every free Google font is hosted on the google api server for public usage. Simply add the following line on top of your css file to import it for immediate use. For example, I want to use the Sofia font, add the following line to your css file:


@import url("https://fonts.googleapis.com/css?family=Sofia");

Now you are good to go! Use it like:

.sofia-font {
font-family: "Sofia";
text-align: center;
}

Import Google's font locally (Marko One)

If you do not have a constant connection to the Google api servers, or running things offline, you will want to download the .woff2 files and have the font saved locally.

1) Find the font you want to use, download the woff2 file.
2) Click on "Try it" from the w3school page, and grab the source URL which looks like:

https://fonts.googleapis.com/css?family=ABeeZee

3) Go to that URL and find source url which looks like:

url(https://fonts.gstatic.com/s/abeezee/v10/yTibYMDoFfqshd01j2jrvwLUuEpTyoUstqEm5AMlJo4.woff2)

4) Go to that location and save the .woff2 file into your local folder
5) Copy the code from the URL in step 2, and paste it into your css file like:

@font-face {
font-family: 'ABeeZee';
font-style: normal;
font-weight: 400;
src: url(fonts/ABeeZee.woff2) format('woff2');
unicode-range: U+0000-00FF, U+0131, U+0152-0153, U+02C6, U+02DA, U+02DC, U+2000-206F, U+2074, U+20AC, U+2212, U+2215;
}

*Notice the red above, the path is wherever you saved the woff2 file

6) Now you are good to go! Use it like:

.abeezee-font {
font-family: "ABeeZee";
text-align: center;
}
